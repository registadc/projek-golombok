**Projek Golombok** adalah sebuah aplikasi web yang dirancang untuk memudahkan pengelolaan dan promosi destinasi wisata. Aplikasi ini menyediakan platform bagi pengelola wisata (admin) untuk mengelola data destinasi, galeri foto, serta informasi penting lainnya, sehingga pengunjung dapat dengan mudah menemukan dan merencanakan perjalanan mereka.

Dibangun dengan **PHP Native** dan didukung oleh **Bootstrap 5**, Projek Golombok menawarkan antarmuka yang responsif, modern, dan mudah digunakan dengan database **MySQL** yang handal.
